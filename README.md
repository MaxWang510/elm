##  项目运行

```bash
# 克隆项目到本地
git clone https://github.com/MaxWang510/elm.git

#  安装依赖
npm install

# 本地开发
npm run dev

#  打包
npm run build

```

##  项目说明

> 用到的技术栈

vue2.0 + vue-router + webpack + vue-cli + axios + stylus + flex
